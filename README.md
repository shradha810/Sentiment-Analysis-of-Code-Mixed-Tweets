# Sentiment-Analysis-of-Code-Mixed-Tweets
SemEval-2020 Task 9 on Sentiment Analysis of CodeMixed Tweets (SentiMix 2020)
1) Aim: To assign sentiment labels for Hinglish tweets.
2) One of the important use case that can be tackled with sentiment analysis is hate-speech detection
3) Code-mixing is one of the norms of multilingual communities where the users primarily transliterate to English language to express their own language.
